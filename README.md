# Student Attendance Management System

A PHP & MySQL web application to manage student attendance with login functionality and attendance reports.

## Features

- Admin login system
- Mark student attendance (present/absent)
- View attendance records
- Search student history
- Database-driven storage

## Technologies

- PHP
- MySQL
- HTML5 & CSS3
- JavaScript
- XAMPP

## How to Run

1. Install XAMPP
2. Copy folder to `htdocs`
3. Create database `attendance_db` in phpMyAdmin
4. Import the database file
5. Update `database.php` with your credentials
6. Start Apache and MySQL
7. Open `http://localhost/attendance-management-system`
